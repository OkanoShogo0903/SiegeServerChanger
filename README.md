# SiegeServerChanger
http://www.ubisoft.co.jp/r6s/
RainbowSixSiege(PC版)のサーバーを簡単に変えるソフトです

## インストール方法、使い方
1.Server_Changer.jarをzipダウンロード&解凍
2.GameSettings.iniのあるファイルにServer_Changer.jarを置く
3.Server_Changerを起動してサーバーの地域を指定
エラーが表示されなかったらうまくいってます
